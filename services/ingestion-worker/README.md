# Ingestion Worker

This directory is reserved for the ingestion and enrichment worker that will process source documents, witness submissions, media artefacts, and downstream analytics jobs.

The ingestion worker is expected to be implemented in Python 3.11 or later to take advantage of mature machine-learning, document-processing, and scientific-computing libraries. Its responsibilities should include source normalisation, classification, enrichment, embedding generation, schema mapping, and tactical analytics workflows such as order-of-battle correlation.
